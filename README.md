# 📊 Sales Dashboard (Excel)

An interactive sales dashboard built using Microsoft Excel to analyze and visualize sales performance.

## 🔍 Overview
This dashboard provides insights into:
- Total Sales
- Sales by Country
- Sales by Managers
- Sales by Representatives
- Quantity Sold

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Data Visualization

## ⚡ Features
- Interactive filters (Year, Manager)
- Dynamic data updates
- Clear and simple visualization
